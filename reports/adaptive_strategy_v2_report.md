### 4. 渐进式参与牛市

```python
def calculate_bull_position(regime_info):
    # 第一层: 识别牛市 (30%仓位)
    # 第二层: 趋势一致 (60%仓位)
    # 第三层: 量能确认 (100%仓位)
    
    base = 0.3
    if regime_info.trend_consistency > 0.7:
        base += 0.3
    if regime_info.volume_confirm > 0.5:
        base += 0.4
    
    return min(base, 1.0)
```
